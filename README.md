## 🚀 Featured Projects

<details>
<summary><b>❤️ Heart Failure Risk Prediction API</b></summary>
<br/>

A machine learning-powered REST API that predicts heart failure risk from 12 clinical parameters, built for fast, reliable, real-time inference.

| Category | Details |
|---|---|
| **Stack** | Python, FastAPI, Scikit-learn, Pandas |
| **Scale** | 12-feature clinical input pipeline, single-record & batch inference |
| **Performance** | Random Forest Classifier with StandardScaler-based preprocessing |
| **Security** | Input validation via FastAPI/Pydantic schemas |
| **Impact** | Demonstrates deployable clinical risk-assessment tooling |
| **Repository** | [Heart_Disease-prediction](https://github.com/abhirammsvyt-ux/Heart_Disease-prediction) |

Trained on the Heart Failure Clinical Records dataset, the model classifies patients as **High Risk** or **Low Risk** based on features like ejection fraction, serum creatinine, and follow-up time. The service exposes interactive Swagger documentation at `/docs`, making it easy to test and integrate into downstream clinical tools.

</details>
